Setup Instruction:
	1. Extract Assignment1.tar.gz
	2. Open Assignment1 folder
	3. type make in terminal shell (to do makefile)	   
	4. Open 3 separate terminal and locate it into Assignment1 folder
	5. Type ./Controller in first terminal (for controller)
	6. Type ./Monitor in second and third terminal (for Monitor1 and Monitor2)
	7. Press <Ctrl-C> in Controller terminal to end the program
